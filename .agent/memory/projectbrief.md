# Project Brief: Orendt Studios Mitarbeiter-Umfrage

## Core Goals
- Create a digital survey app for anonymous employee feedback.
- Provide an admin dashboard for question management and result analysis.
- Maintain a premium, modern "Orendt Studios" branding (Black/White with Accent).

## Scope
- Step-by-step mobile-friendly survey form.
- Admin dashboard (`/admin`) with question CRUD operations.
- Data persistence using Supabase.
- Deployment on Vercel.
